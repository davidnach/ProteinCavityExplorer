About Protein Cavity Explorer:

Protein Cavity Explorer is a web tool that allows a user to input any protein ID and explore the cavities that the protein possesses.

Protein Cavity Explorer was developed by as a Senior Project at Western Washington University in Fall 2017 under the advisement of Filip Jagodzinski.

Protein Cavity Explorer uses PV(Protein Viewer) and D3 to create histograms. The site was done in Angular 5.

ProCavEx displays a visualization of the protein structure where you can select a pocket of the protein's A chain and see the residues that exist within that pocket. To the right of the structure you can also select multiple pockets and compare the types of residues that exist within all the pockets selected.

At the moment it only works for proteins in this list.
