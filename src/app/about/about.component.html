About Protein Cavity Explorer:

Protein Cavity Explorer is a web tool that allows a user to input any protein ID and explore the cavities that the protein possesses.

Currently there is a visionalization of the protein structure where you can sselect a pocket on protein and the residues that exist within that pocket will go red and become spheres. To the right of the structure you can also select multiple pockets and compare the types of residues that exist within all the pockets selected.

At the moment it only works for proteins in this list. And only for A chains.
